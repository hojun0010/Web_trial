문제 : scroll_sidebar와 document_head가 서로 붙어 있지 않는다
        => 창 크기 조절 및 확대 축소 시 중간에 빈 공간이 생기는 문제 발생
        -> 해결 : css에서 document_head 부분이 갖고 있던 position : fixed 속성을 제거 
        - fixed 속성을 갖고있는 요소는 웹사이트 상에서 다른 요소와 상관없이 영역을 갖는다 따라서
         이 영역은 다른 요소에 의해 침범될수있게 된다.


문제 : 확대 축소 및 창 크기 조절 시 실제 유튜브 웹페이지 처럼 동영상 배치 방법이 바뀌도록 한다.
